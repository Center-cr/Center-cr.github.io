第一步

关闭mysql服务，可以在cmd中关闭（命令：net sopt mysql），也可以直接到服务中关闭。

或者使用win+R services.msc

第二步

以管理员身份运行cmd，这一步需要注意，一定是管理员身份下运行，否则后面则无法进行哦

第三步

越权。进入mysql的安装目录bin目录下，

报错：

备份图片: D:_Note48d4e5cea40ea750f4b1cb4ae59363.png

Powershell 中 报错Set-Location : 找不到接受实际参数

解决方法： 在Program Files 和 MySQL Server 8.0 两边加上英文双引号"" C:“Program Files”"MySQL Server 8.0"

原因：路径存在空格

输入命令： mysqld --console --skip-grant-tables --shared-memory ，直到看到显示 “mysqld: ready for connections.”

输入. .--console --skip-grant-tables --shared-memory

启动失败

输入.--initialize-insecure

第四步

登录mysql。打开新的cmd界面（非管理员身份即可）到mysql的bin目录下，输入“mysql -uroot -p”登录mysql，弹出需要输入密码，直接回车即可。

第五步

查看数据库表“show tables;” ，此时会看到用户表为user，查看user表结构“show columns from user;”
